[首页][blog_link] **我的linux配置** # 我用的是manjaro+i3wm+polybar 1.[manjaro][manjaro_link]是一个专业制作的基于Linux的操作系统，是Windows或MacOS的合适替代品。多种桌面环境可通过我们的官方和社区版本。我们还与制造商合作设计专用硬件。 2.[i3wm][i3_link]是一个平铺式的窗口管理器 平铺:平铺就是之所有的窗口都不会相互重叠，而是"自动的"被调整大小,使得它们能够刚好占满整个屏幕,使屏幕空间得到充分利用. 3[polybar][polybar_link]是一个快速,容易使用的状态栏. ![我的polybar](./图片/polybar) [我的polybar配置](./polybar_config/config) [双显示器polybar启动脚本](./polybar_config/launch.py) # 我的linux 配置更新了 ps ： 忘了更新了 我用的是arch+i3wm+polybar - 简要说一下arch+i3的引导配置吧， 防止忘了 - arch 装 xorg xorg-xinit 当然还有i3 - 在~/.xinitrc 配置: ~~~~~~~~~~~~~~shell (base) xtcc@xtccarch:~/small_parking.github.io (master *) $ cat ~/.xinitrc export GTK_IM_MODULE=fcitx5 export QT_IM_MODULE=fcitx5 export XMODIFIERS="@im=fcitx5" # 前三个是输入法的配置 export LANG="zh_CN.UTF-8" export LC_CTYPE="zh_CN.UTF-8" exec i3 # 这个是启动i3的配置 ~~~~~~~~~~~~~~~~~~~~~~~ ~~~~~~~~~~~~~~~~~~~~~~~shell cat ~/.zprofile # 放到用户目录下 达到登录即启动i3 startx ~~~~~~~~~~~~~~~~~~~~~~~~~~~ [blog_link]:https://xtccc.github.io/small_parking.github.io/ [polybar_link]:https://github.com/polybar/polybar [i3_link]:https://i3wm.org/ [manjaro_link]:https://manjaro.org/
